## Getting Started

1. **Clone the Repository:**

   ```bash
   git clone https://github.com/saadazghour/Local-Restaurant-Finder.git
   cd Local-Restaurant-Finder
   ```

2. **Install Dependencies:**

   ```bash
   npm install
   ```
